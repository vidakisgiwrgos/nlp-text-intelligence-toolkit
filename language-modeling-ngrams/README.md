# N-gram Language Modeling (Bigram/Trigram)

This module implements classic statistical language models (bigrams/trigrams) with:
- probability estimation & smoothing
- evaluation via cross-entropy / perplexity
- practical text completion utilities
- context-aware spelling correction (edit-distance based)

> Academic origin: MSc NLP course project. Presented here as a portfolio module.
