# Attention BiRNN Text Classifier (GRU/LSTM)

Text classification pipeline using a stacked bi-directional RNN (GRU/LSTM) with self-attention.

## What’s included
- Strong baselines (majority / logistic regression / MLP)
- BiRNN (GRU/LSTM) with self-attention
- Hyperparameter tuning (layers, hidden size, dropout)
- Training vs development curves (loss over epochs)
- Precision/Recall/F1 and PR-AUC evaluation

## Files
- `docs/` – specification + report (reference)
- `src/` – implementation (.py)
- `notebooks/` – notebook demo (.ipynb)
