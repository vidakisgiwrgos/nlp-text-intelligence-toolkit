# BiRNN POS Tagger

Sequence labeling pipeline that assigns POS tags using a stacked bi-directional RNN.

## What’s included
- POS tagging baseline + neural model comparison
- BiRNN model training & evaluation
- Development monitoring (loss curves)
- Dataset statistics and preprocessing notes

## Files
- `docs/` – links to the shared spec/report
- `src/` – implementation (.py)
- `notebooks/` – notebook demo (.ipynb)
