# POS Tagging with Word Embeddings + MLP

Sequence labeling pipeline that maps tokens to POS tags using pretrained word embeddings and a Multi-Layer Perceptron (MLP).

## What’s included
- UD treebank parsing (CoNLL-U)
- Word embedding lookup (Word2Vec)
- MLP tagger training & evaluation

## Files
- `docs/` – specification + report
- `src/` – implementation in `.py`
- `notebooks/` – optional demo notebook
