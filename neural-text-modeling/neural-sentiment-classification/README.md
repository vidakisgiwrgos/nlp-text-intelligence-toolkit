# Neural Sentiment Classification (Baseline vs MLP)

End-to-end text classification pipeline comparing classic baselines to a Multi-Layer Perceptron (MLP).

## What’s included
- TF-IDF features (n-grams)
- Baseline model (Logistic Regression)
- Neural model (MLP in PyTorch)
- Learning curves and evaluation analysis

## Files
- `docs/` – specification + report
- `src/` – implementation in `.py`
- `notebooks/` – optional demo notebook
