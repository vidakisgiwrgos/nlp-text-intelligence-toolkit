# BERT POS Tagger (Token Classification Fine-Tuning)

Fine-tuned a BERT model for POS tagging as a token classification task.

## What’s included
- Token/label alignment for token classification
- Fine-tuning with Hugging Face `Trainer`
- Metrics: accuracy + macro F1 (as implemented)
- Baseline references (most frequent tag strategy)

## Files
- `docs/` – links to shared spec/report
- `src/` – implementation (.py)
- `notebooks/` – notebook demo (.ipynb)
