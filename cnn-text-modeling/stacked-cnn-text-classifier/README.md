# Stacked CNN Text Classifier (n-gram filters + residuals)

Text classification pipeline using a stacked CNN encoder with n-gram filters and global max pooling.

## What’s included
- Baselines: majority / logistic regression / MLP / RNN comparison (as reference)
- Stacked CNN with configurable filters and kernel sizes (n-gram modeling)
- Development monitoring (loss/accuracy/F1 over epochs)
- Hyperparameter tuning (dense dimension, filters, kernel length)
- Precision/Recall/F1 and PR-AUC evaluation

## Files
- `docs/` – specification + report (reference)
- `src/` – implementation (.py)
- `notebooks/` – notebook demo (.ipynb)
