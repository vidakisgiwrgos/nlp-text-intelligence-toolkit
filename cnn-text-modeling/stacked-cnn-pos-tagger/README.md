# Stacked CNN POS Tagger (sequence labeling)

Sequence labeling pipeline for POS tagging using a stacked CNN encoder.

## What’s included
- Strong baseline reference (most-frequent tag per word)
- Stacked CNN with n-gram filters + residual connections
- Development monitoring during training
- POS tagging evaluation and model comparison

## Files
- `docs/` – links to shared spec/report
- `src/` – implementation (.py)
- `notebooks/` – notebook demo (.ipynb)
